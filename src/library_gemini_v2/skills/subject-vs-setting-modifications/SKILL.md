# Subject vs. Setting Modifications

When a prompt requests a change in setting (e.g., 'Make it look like a sold out concert', 'Change background to a grassy savannah'), the main subject MUST be preserved.

## Guidelines
1. **Subject Preservation is Paramount**: If Image B creates a perfect new setting but distorts, replaces, or destroys the original main subject (e.g., the person, the polar bears), it must be heavily penalized.
2. **Accepting Imperfect Settings**: Image A is preferred if it preserves the subject perfectly and makes a reasonable (even if less dramatic or slightly studio-like) attempt at the setting change, compared to an image that nails the setting but ruins the subject.
3. **Check for Collateral Damage**: Always verify that the subject's identity, pose, and key attributes remain intact after a background or setting transformation.
