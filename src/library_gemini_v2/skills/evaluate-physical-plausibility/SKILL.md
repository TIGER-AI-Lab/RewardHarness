# Evaluate Physical Plausibility

When an instruction requires inserting a new object or replacing an existing one (e.g., replacing a remote with a bottle, inserting a person):

1. **Check for Floating Objects**: Ensure the new object is grounded. Objects should not appear to be floating in mid-air above tables, floors, or hands unless explicitly requested.
2. **Gravity and Interaction**: The object must obey basic physics. If it is replacing an object held by a person, it should fit within the grasp or rest naturally on the surface.
3. **Severe Integration Failures**: While minor blending issues are acceptable, a completely floating object or one that defies gravity is a severe failure in physical plausibility and should be heavily penalized, even if the object itself is generated perfectly.
