# Evaluating Text Additions

1. **Exact Match is King**: The primary criterion for text additions is whether the exact requested string is present and spelled correctly. Pay attention to capitalization and punctuation if specified.
2. **Ignore "Subtle Integration"**: Do NOT penalize text for looking "abrupt", "artificial", or "digitally added". An image with a clear, correct, but artificial-looking text addition is vastly superior to an image with a "natural and subtle" text addition that is misspelled, blurry, or hard to read.
3. **Placement**: Verify the text is in the requested location (e.g., "on the black band", "bottom left").
