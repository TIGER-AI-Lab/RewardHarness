# Evaluating Background Changes
When a prompt asks to change the setting (e.g., "grassy savannah", "sold out concert", "elementary school cafeteria"):
1. **Foreground Preservation**: The main subjects (e.g., polar bears, people) MUST remain intact and unchanged unless specified.
2. **Background Accuracy**: Does the new background clearly represent the requested scene? Look for specific contextual clues (e.g., stage lighting and crowds for a concert, trays and tables for a cafeteria).
3. **Lighting and Shadows**: Do the lighting and shadows on the foreground subjects match the new background? Mismatched lighting makes the edit look like a cheap cut-and-paste job.
