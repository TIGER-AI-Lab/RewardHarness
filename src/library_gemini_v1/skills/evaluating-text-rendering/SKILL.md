# Evaluating Text Rendering
When evaluating edits that involve adding or modifying text, check the following:
1. **Spelling**: Does the text exactly match the prompt? Watch out for typos (e.g., "beging" instead of "begging"). A spelling error is a major failure in instruction following.
2. **Placement**: Is the text located exactly where requested (e.g., "bottom left")?
3. **Style/Color**: Does the text match the requested color, font, or style?
4. **Integration**: Does the text look naturally integrated, or is it distorted, stretched, or blurry?
5. **Unintended Changes**: Did the model alter other parts of the image while adding the text?
