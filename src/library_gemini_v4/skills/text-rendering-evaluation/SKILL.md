# Text Rendering Evaluation
When evaluating prompts that request adding or modifying text:
1. **Exact Spelling (Critical)**: Check the spelling letter-by-letter. If the prompt asks for 'begging', and the image says 'gegging' or 'beging', it is a severe failure in Instruction Following.
2. **Typography and Integration**: Assess if the font style, color, and placement match the prompt.
3. **Artifacts**: Look for blurry text, weird character artifacts, or floating letters that degrade visual quality.
